# 🚀 Deploy na Railway

Este projeto está configurado para deploy na Railway.app com 23 coleções:

## Coleções incluídas:
1. curso, evento, noticia, aviso (existentes)
2. 19 novas coleções EQAVET e administrativas

## Como fazer deploy:

1. **Acesse:** https://railway.app
2. **Login com GitHub**
3. **"New Project" → "Deploy from GitHub repo"**
4. **Selecione este repositório**
5. **Railway faz TODO automaticamente!**

## URLs após deploy:
- Admin: https://seu-projeto.railway.app/admin
- API: https://seu-projeto.railway.app/api
